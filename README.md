# advent of code 2018

## Project setup

```
npm install
```

### Running a given day

```
npm start [day number]
```

The day number should be between 1 and 24. To run part two of a day, simply add .5 after the number.

For example, if I wanted to run day 4 part 2, I would run `npm start 4.5`
